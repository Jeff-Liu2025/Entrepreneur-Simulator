## 📝 使用建议

### 生成工具适配

- **Midjourney**：直接复制英文提示词，末尾加 `--ar 16:9` 或 `--ar 1:1` 控制比例
- **Stable Diffusion**：英文提示词 + 负面词 `blurry, smooth, gradient, realistic, 3d`
- **DALL-E / 文心一格**：中文描述更准确，可直接用中文描述生成

### 后期处理

- 如果生成的图不够像素化，可用 Photoshop「滤镜→像素化→马赛克」或在线工具 `pixelateimage.com` 处理
- 统一调整色盘到游戏主色调（黄 #f4d03f / 蓝 #5d8aff / 绿 #4ade80 / 红 #ff4d6d）
